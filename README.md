# FILKOMEVENT

FILKOMEVENT adalah aplikasi web pusat informasi digital acara FILKOM yang dirancang untuk menjadi platform terpusat bagi pengelolaan, publikasi, dan akses informasi event di lingkungan Fakultas Ilmu Komputer Universitas Brawijaya.

Sistem ini digunakan untuk memudahkan mahasiswa dalam menemukan informasi acara secara cepat, jelas, dan terstruktur, tanpa harus mencari dari berbagai media yang terpisah. Melalui website ini, pengguna dapat melihat daftar event, membaca detail acara, melakukan pendaftaran secara online, serta memantau status partisipasi mereka pada event yang diikuti.

Di sisi pengelolaan, sistem juga mendukung kebutuhan admin dalam mengatur data event, memperbarui informasi acara, mengunggah poster, mengelola peserta, serta memantau jalannya proses pendaftaran. Dengan adanya FILKOMEVENT, proses penyebaran informasi event menjadi lebih terpusat, tertata, dan efisien bagi seluruh pihak yang terlibat.
